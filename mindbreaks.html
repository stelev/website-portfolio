MindBreaks: The Human Recharge App

MindBreaks is a program with an app which delivers a “human recharge station” via a suite of meditations by Deepak Chopra. I designed the user experience for the app and worked from low fidelity to high fidelity wireframes to deliver the user interface for the app.

Project goals: 1. Guide the user through the MindBreaks experience and set up 2. Provide an easy to use media player for the meditations 3. Receive data on user experience
